为下一个 session 生成启动 prompt。

## 做什么

回顾当前对话，提取：
- 讨论确定要做但未完成的事项
- 达成共识的方案（引用相关文件路径）
- 需要改的关键文件

直接输出，不读任何项目文件。

## 输出格式

```
## 项目: [从 cwd 取项目名]

## 待办
[从对话中提取，按优先级排列]

## 方案和关键文件
[已讨论的方案要点 + 相关文件路径]

## 注意事项
[踩过的坑、特殊约定，没有就省略这段]
```

## 规则

- 不读文件、不跑命令，纯从对话上下文提取
- 没有明确待办时直接告诉用户"本次对话没有未完成的待办"，不要硬凑
- 用中文
