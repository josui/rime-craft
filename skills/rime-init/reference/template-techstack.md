# Techstack 模板 `[开发]`

技术选型、项目结构、阶段计划。

## 模板

```markdown
# {项目名} — 技术选型 & 项目结构

## 技术栈

| 用途 | 选型 | 说明 |
|------|------|------|

## 项目结构

<!-- 目录树 + 行内注释 -->

## 依赖清单

<!-- 列出所有直接依赖及用途 -->

## 开发阶段

### Phase 1: xxx ✅
- [x] 已完成项
- [ ] 待做项

### Phase 2: xxx
- [ ] ...
```

## 写法要点

- 目录树要及时更新，新增文件/目录时同步
- Phase checklist 完成项打 `[x]`，新阶段直接追加
- 依赖清单注明每个依赖的用途，不只列包名
