# Archive 模板

已完成功能的归档。阶段全部完成后从 PRD 整体移入。

## 模板

```markdown
# {项目名} Archive — 已完成功能

> 从 PRD 归档的已完成功能。保留记录，不再活跃管理。

## P0 — MVP（全部完成）

<!-- 可选：保留该阶段的 scope 决策说明 -->
<!-- > 例：基于"一个月可用版"的时间约束，i18n 延后... -->

| #    | 内容 | 状态 | 说明 |
| ---- | ---- | ---- | ---- |
| #001 | xxx  | ✅   | xxx  |
| #002 | xxx  | ✅   | xxx  |
```

## 写法要点

- 按阶段分节（P0、P1...），可附带该阶段的 scope 决策 blockquote
- **编号不变**，从 PRD 原样搬过来
- 只有全部 ✅ 的阶段才归档，部分完成的留在 PRD
